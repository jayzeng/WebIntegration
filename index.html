Knock Web Integration Library

First, initialize the library with your company id:

knock.init('company id'); // initialize the library with your company ID

There are 4 ways to integrate Knock into your website:

1) An iframe that is embedded into the page. View demo

2) Open a modal iframe using a button or link on your site which links to your Knock company profile. View demo

knock.open(); // opens a modal window containing your company profile
3) Open a modal iframe using a button or link on your site which links to a specific listing. View demo

knock.open('listing id'); // opens a modal window containing the specified listing

4) Open a modal iframe using a button or link on your site which links to a slimmed down scheduling interface View demo

knock.openScheduling(); // opens a modal window containing a slimmed down scheduling view with all listings/floorplans

If you wish to embed our UI directly into your website, you can use the following URL scheme within an iframe:

http://knockrentals.com/company/<company_id>?isExternal=true&companyName=YOUR_COMPANY_ID_HERE
